# MMC Diploma in CST - 7th Semester Management System

## Project Overview

I am a student of **Madan Mohan College**, pursuing **Diploma in Computer Science & Technology (CST)** in the **7th semester**. As part of my **Project Work-II** course, I have developed this complete **Course Management System**.

This project was created with the help of **Vibe Coding** - an innovative approach combining modern development tools with AI assistance to build functional and professional web applications efficiently.

## Why I Built This

As a 7th semester CST student, I needed a practical project that would demonstrate my understanding of web development technologies while also creating something useful for educational institutions. This system solves real problems faced by teachers and administrators in managing course-related data.

## Features

- 📊 **Dashboard** - Quick overview with statistics, top students, and recent assignments
- 👨‍🎓 **Student Management** - Add, edit, delete and view student profiles
- 📚 **Subject Management** - Complete 7th semester subjects with all details
- 📅 **Schedule Management** - Weekly timetable with periods and room allocation
- ✅ **Attendance Management** - Track attendance with filters and summary
- 📈 **Marks Management** - Record and calculate marks with grading system
- 📝 **Assignment Management** - Track assignments with status
- 📁 **Resource Management** - Upload and organize study materials

## Technology Stack

- **HTML5** - Structure and markup
- **CSS3** - Modern styling with responsive design
- **JavaScript (ES6)** - Complete functionality and data management
- **Font Awesome** - Icons for better UI experience

## How to Use

1. Open `index.html` in any modern browser
2. Navigate through tabs using the sidebar
3. Use "Add" buttons to insert new data
4. Edit or delete existing records
5. Track attendance and marks easily
