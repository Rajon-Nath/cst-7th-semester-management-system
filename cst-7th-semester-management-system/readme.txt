# 🎓 Diploma in CST - 7th Semester Management System

A comprehensive semester management system for Diploma in Computer Science & Technology, 7th Semester.

## 📋 Overview

This is a complete management system designed specifically for CST 7th semester with all subjects, schedules, attendance tracking, marks management, assignments, and resources.

## 🚀 Features

### 1. Dashboard
- Quick stats (Subjects, Periods, Credits, Marks)
- Subject progress bars
- Weekly schedule overview
- Recent assignments

### 2. Subjects Management
- View all 7 subjects with details
- Add new subjects
- Edit existing subjects
- Delete subjects

### 3. Schedule
- Weekly timetable with all periods
- Subject and room allocation
- Break periods

### 4. Attendance
- Track attendance per subject
- Percentage visualization with progress circles
- Quick mark attendance feature
- Filter by subject

### 5. Marks
- Marks for each subject
- Total marks, obtained, percentage
- Grade calculation
- Pass/Fail status

### 6. Assignments
- View all assignments
- Status tracking (Pending/Submitted/Graded)
- Add new assignments

### 7. Resources
- Access study materials
- File type and size display
- Download/Share options

## 🛠️ Technologies

- HTML5
- CSS3 (Custom properties, Grid, Flexbox)
- Vanilla JavaScript (ES6 Classes)
- Font Awesome Icons
- No external dependencies
